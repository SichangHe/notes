# Machine Learning

## general problem format

given input $A$, want output $\vec y\in Y$

$$
f:A\rightarrow Y
,\qquad
\vec y=f(a)\in Y
\quad\text{for }a\in A
$$

- traditional approach\
    hand-craft the function $f$
- machine learning\
    build another function $\phi$, feed it data $L$
