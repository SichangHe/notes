# Differential Equation

an equation that contains an unknown function and one or more of its derivatives.

- order\
    highest derivative
- solution\
    function that satisfy the equation
- initial condition
- initial value problem

## direction field

$$
y_{n+1}=y_n+hy'_n
$$
